¿Cómo agrego una pieza nueva? (sin tocar código)

Abrí data.json en GitHub (ícono del lápiz), agregá un objeto nuevo dentro del array posts con esta forma, y hacé commit. El dashboard se actualiza solo la próxima vez que lo abras.

{
  "id": 13,
  "fecha": "2026-07-10",
  "titulo": "Hook de la pieza",
  "formato": "carrusel",
  "angulo": "anti-mito",
  "keyword": "DIAGNOSTICO",
  "instagram_media_id": null,
  "media_product_type": null,
  "resultado_manual": "ganador",
  "metricas": {
    "vistas": 12000,
    "alcance": 15000,
    "guardados": 340,
    "comentarios_totales": 210,
    "comentarios_keyword": 58,
    "compartidos": 90,
    "dms_iniciados": 40,
    "calls_agendadas": 3
  },
  "notas": "lo que quieras anotar"
}

Valores posibles de formato: carrusel, video, no producido, no especificado.
Valores sugeridos de angulo: anti-mito, autoridad-educativo, autoridad-diagnostico, prueba-social, opinion-fuerte, protocolo-practico, historia-personal, producto-formacion.
Valores sugeridos de resultado_manual: ganador, muy bueno, bueno, no pego, sin feedback, no producido todavía.

Auto-sync con Instagram: si completás instagram_media_id (el ID del post/reel) y media_product_type ("REELS" para reels, dejalo vacío para el resto), el robot diario le va a pisar los campos de metricas con los números reales de Instagram — no hace falta que los cargues a mano para esas piezas. Corré node scripts/sync-instagram.js --list para ver los IDs de tus posts recientes.
